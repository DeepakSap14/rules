Localization Framework is based on '@ngx-translate/core', where we have custom loader that supports among other things pseudolocalization as well as having a comments so you know something more about the localized field. Resource files are saved under /assets/i18n/

Simple translate using a PIPE. It searches inside assets/i18n/lang/resource.app.json:
{{ 'DEMO.hello_word' | translate }}
Translating text paragraph
{{ 'DEMO.info_paragraph' | translate }}
Localized string can accepts also parameters:
{{ 'DEMO.hello_word_with_param' | translate:{value: myParam} }}
Dates can be formatted based on the the locale. For more formatting patters please see: assets/i18n/lang/resource.widgets.date.json. In this example we are using our custom locale aware Pipe formatTo:
{{ 'DEMO.hello_word_with_param' | translate:{value: myDate| formatTo: 'shortDate' } }}
Currency can be formatted based on the the Language Locale and Currency Code.
Formatting an currency:
{{ 'DEMO.hello_word_with_param' | translate:{value: myCurrency| formatTo: 'currency' } }}
